Shortest Path Robot

The shortest path robot recieves a target destination through a webpage and uses Dijkstra's shortest path algorithm to navigate there as quickly as possible.The webpage takes an input of a letter, which represents a specific intersection on the course. Reflective sensors are used to ensure the robot stays on the lines and a wifi chip is used to recieve instructions from the database. My role on this project focused on structuring and writing the arduino code (C/C++) so the robot could handle different situations. The robot moved through different states depending on what event occured. For example, one state was of course driving. When the robot entered an intersection it needed to decide to go straight, left, right or make a U turn. This was determined from the output of dikjstra's algorithm, which was the order of intersections to progress through. It was then necessary to translate this list of letters into directions for the robot so the correct turns were carried out at intersections.

Smart Cat Feeder

The Smart Cat Feeder detects, dispenses, and presents food to the appropriate cat. Once the correct cat is detected through the use of RFID, a tray holding the food becomes available. The inspiration for this project is a multi-cat household in which the cats do not eat the same type of food. The entire product development cycle was utilized during the course of this project. Core requirements were defined from user engagement data and market research. Additionally, many rounds or prototyping occured. Starting with user experience prototypes of individual components and ending with the integration of all subsystems. In order to construct a functional system, we 3D printed dispenser components, used RFID for detection, and built a retractable tray with a re-purposed gear belt and stepper motor.

Compressible Flow Simulations for Shock/Expansion Tube Performance Analysis

I am currently performing reserach at the UVA Aerospace Reserach Laboratory on the shock/expansion tube facility. My research focuses on evaluating the capabilities of the hypersonic wind tunnel that could be used to test spacecraft and high speed aircraft. This is being performed through the installation of computational flow software. Additionally, I am writing simulation scripts based off the physical properties of the wind tunnel to evaluate the performance envelope. Simulation values of speed and pressure will be compared to values from physical testing.

Follow the leader robot

The follow the leader robot uses PID control to stay a set distance behind the leader robot. It uses reflective sensors to stay on the line, IR sensors to recieve turn commands and ultrasonic sensors to determine its following distance.
